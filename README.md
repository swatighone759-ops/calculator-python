# Calculator Project

## 📌 Description
This is a simple calculator program written in Python.  
It allows users to perform basic arithmetic operations.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Run the Python file
2. Enter two numbers
3. Choose operation (+, -, *, /)
4. Get result

## 📷 Example
Enter first number: 10  
Enter second number: 5  
Choose operation (+, -, *, /): +  
Result: 15
